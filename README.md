# CRM ویندوزی فارسی

این پروژه یک CRM دسکتاپی با پایتون است که برای ویندوز طراحی شده و داده‌ها را در فایل‌های CSV ذخیره می‌کند.

## امکانات

- پروفایل اختصاصی برای هر کاربر ویندوز
- ثبت، ویرایش و حذف نامحدود مشتری
- ثبت، ویرایش، حذف و دوبارکلیک برای باز کردن معامله
- یادآور معامله با تاریخ، ساعت، اعلان ویندوز و صدای بیپ
- ۷ دسته‌بندی پیش‌فرض قابل ویرایش
- کاریزهای فروش قابل ویرایش
- داشبورد با فیلتر دسته‌بندی
- بخش اعلان‌ها
- بخش فروش و درآمد
- رتبه‌بندی کارشناسان فروش بر اساس معاملات موفق
- ذخیره‌سازی کامل در CSV

## اجرا

```powershell
python -m pip install -r requirements.txt
python main.py
```

## محل ذخیره داده‌ها

داده‌ها داخل پوشه `data` در همین پروژه ذخیره می‌شوند. برای هر کاربر ویندوز و هر پروفایل، CSVهای جدا ساخته می‌شود.
